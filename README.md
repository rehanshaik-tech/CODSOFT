# Recommendation System

## Overview

This project is a simple Recommendation System developed in Python. It recommends movies to users based on their preferred genre using the **Content-Based Filtering** technique.

The system asks the user to enter a favorite genre and then displays a list of movies that belong to that genre.

## Features

* Simple and easy-to-use interface
* Content-based movie recommendations
* Beginner-friendly Python implementation
* No external libraries required
* Fast and lightweight

## Technologies Used

* Python 3

## Project Structure


## How It Works

1. Run the Python program.
2. Choose your favorite movie genre.
3. The system searches its movie database.
4. Movies matching the selected genre are recommended.

## Available Genres

* Sci-Fi
* Action
* Horror
* Comedy

## Sample Output

```

2. John Wick

```

## Recommendation Technique

This project uses **Content-Based Filtering**, where recommendations are made by comparing the user's preferred genre with the genres stored in the movie dataset.
